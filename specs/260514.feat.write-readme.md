## 现状分析

### 项目概况

DimCut 是一个基于 SolidJS 的**视频时间轴编辑器组件库**，核心创新是将传统一维时间轴折叠为**二维多行布局**，并将 ASR 语音转写文字与视频缩略图、音频波形对齐排列，实现"文字驱动视频剪辑"的交互范式。

### 技术栈

| 类别 | 技术 |
|------|------|
| UI 框架 | SolidJS（细粒度响应式） |
| 样式 | TailwindCSS v4 |
| 语言 | TypeScript |
| 多媒体处理 | mediabunny（视频解码/音频解码/编码导出） |
| 构建 | Vite（Library Mode） |
| 测试 | Vitest |
| 包管理 | pnpm |

### 项目输出

1. **组件库**（`src/lib/`）：以 Vite Library Mode 构建为 ES module，导出 `Timeline`、`PreviewPlayer`、`exportVideo` 等组件和函数。
2. **Landing 页面**（`src/landing/`）：项目官网，包含 Hero、Concept（理念介绍）、DemoSection（在线演示）等模块，支持中英双语（i18n）。
3. **Demo 应用**（`src/demo/`）：组件库使用示例。

### 3. GitHub Pages 部署方案

**方案：在 vite.config.ts 中通过环境变量切换构建模式**
