   - `Timeline` 组件通过 `props.currentTime` 接收，通过 `props.onSeek` 回调时间跳转
   - `PreviewPlayer` 通过 `onTimeUpdate` 持续更新 currentTime（播放时）

2. **ASR 组件结构**
   - `AsrTrack`（`src/lib/AsrTrack.tsx`）渲染 ASR 文字，每个 word 是一个 `<span>`
   - 每个 word span 已有 `data-asr-word-start` 和 `data-asr-word-end` data 属性
   - 当前不接收 `currentTime`，无法判断当前高亮状态

3. **TimelineRow 组件**
   - `TimelineRow`（`src/lib/TimelineRow.tsx`）是每个行的容器
   - 已有 `onRangeSelectStart` 等 mouse 事件处理

   - 不接收 `currentTime` 或 `onSeek`

4. **时间指示器（playhead）**
   - `Timeline.tsx:486-536` 中已实现红色竖线指示器
   - 使用 `isInRow()` 判断 currentTime 是否在当前行范围内（`row.startTime <= time < row.endTime`）
   - 此模式可作为行级判断的参考

5. **数据流路径**
   - App → Timeline (currentTime, onSeek) → TimelineRow → AsrTrack
   - 当前 `TimelineRow` 未将 `currentTime`/`onSeek` 传递给 `AsrTrack`

## 技术实现方案

### 方案概览

在现有数据流中透传 `currentTime` 和 `onSeek`，在 `TimelineRow` 层做行级时间区间判断（性能优化），在 `AsrTrack` 中实现 word 级高亮和点击跳转。

### 步骤 1：扩展 AsrTrack props

在 `AsrTrackProps` 中新增：
- `currentTime?: number` — 当前播放时间
- `onSeek?: (time: number) => void` — 点击文字时的跳转回调

### 步骤 2：扩展 TimelineRow props

在 `TimelineRowProps` 中新增：
- `currentTime?: number`
- `onSeek?: (time: number) => void`

将这两个 props 传递给 `AsrTrack`，但仅在行级时间区间判断为真时传入 `currentTime`，否则传 `undefined`（避免不必要的 word 级计算）。

判断逻辑：`currentTime >= row.startTime && currentTime < row.endTime`

### 步骤 3：AsrTrack 中实现高亮逻辑

- 接收 `currentTime`（可能为 undefined，表示当前行不包含播放时间）
- 为每个 word span 添加条件样式：
  - 当 `currentTime >= w.start && currentTime < w.end` 时，应用 `bg-[#bbb] rounded-[2px] p-[2px]`
  - 否则保持原样式
- 添加 `onClick` 事件，调用 `onSeek?.(w.start)` 跳转到当前词的开始时间

### 步骤 4：Timeline 传递 props

在 `Timeline.tsx` 中，为每个 `TimelineRow` 传递 `currentTime` 和 `onSeek`：
- `currentTime={props.currentTime}`
- `onSeek={props.onSeek}`

### 性能优化策略

- 行级短路：`TimelineRow` 先判断 `currentTime` 是否在 `[row.startTime, row.endTime)` 范围内，不在范围内则传 `undefined` 给 `AsrTrack`
- `AsrTrack` 接收 `undefined` 时，所有 word 均不高亮，避免逐 word 比较
- 使用 Solid.js 的响应式系统，仅在 `currentTime` 变化时才重新计算高亮状态

### 文件改动清单

- `src/lib/types.ts` — 无需改动（TimelineProps 已有 currentTime 和 onSeek）
- `src/lib/AsrTrack.tsx` — 扩展 props，添加高亮样式和点击事件
- `src/lib/TimelineRow.tsx` — 扩展 props，添加行级判断，透传给 AsrTrack
- `src/lib/Timeline.tsx` — 传递 currentTime 和 onSeek 给 TimelineRow
