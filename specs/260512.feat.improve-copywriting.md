# 优化首页文案

## 背景

- 基本完成了落地页的样式与交互 @src/landing/
- i18n 配置 @src/i18n/locales/

## 需求

- 需要优化文案，替换或优化已有的文案：
- hero
  - 文本优先的视频时间轴 -> 二维时间轴
  - 像读剧本一样编辑时间轴 -> 突破一维限制，视频一览无余
  - 为播客与口播视频设计的多行折叠时间轴，用阅读取代横向拖动；所有处理在浏览器本地完成。 -> 全新的交互体验：文字、声音、画面，多维信息高度整合，离线网页 APP，免费、隐私、便捷触手可及。
- illust
  - 第一行：将一维时间轴折叠，与文字一起二维排列
  - 第二行：复制、~~删除~~、重排文字，等同于编辑视频
- concept
  - 重新设计 concept 的文案内容，体现体验效率方面的优势：
    - 二维时间轴相比一维时间轴中，有更高的信息密度，能一次看到更多信息
    - 二维时间轴能避免传统一维时间轴，频繁进行向拖动、缩放
    - 文字、画面（视频帧）、声音（波形图）多维信息整合，更方便信息查找、编辑
    - 特别适合知识类视频、博客场景
  - 可以保持当前样式与结构
- demo
  - 用真实视频试试。 就在这里。 -> 立即体验，可替换成你的本地文件。
