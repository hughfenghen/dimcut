Core Features
为长视频剪辑而生
每一个特性都围绕「在有限屏幕内高效编辑长内容」这一核心命题展开。

多行折叠时间轴

长视频自动折叠为多行显示，在有限宽度内纵览全局。跨行片段智能切割，层级自动装箱排列。

ASR 文字轨整合

语音识别文本与时间轴深度绑定。词级高亮跟踪播放进度，点击跳转，删除区间自动标记对应文字。

区间删除编辑

框选或精确定义删除区间，自动合并重叠范围。导出时无缝跳过删除片段，时间戳自动修正。

音频波形可视化

一次性解码 + rawPeaks 缓存 + 按需降采样。5 分钟视频的波形从 20 秒加载优化至毫秒级。

视频缩略图提取

IntersectionObserver 驱动懒加载，仅提取可视区域内帧。批量缓存 + 分步渲染，滚动始终流畅。

流式视频导出

StreamTarget 逐帧写入文件，避免内存溢出。有效片段自动拼接，大文件导出稳定可靠。

Signature Feature
折叠，而非滚动
传统时间轴以水平滚动应对长内容，编辑者永远只能看到一小段。 DimCut 将时间轴像书页一样折叠——每一行都是一段连续的时间窗口， 跨行的片段被智能切割为可见切片。

Deep Integration
文字即时间轴
ASR 文字轨不是附属品，而是时间轴的一等公民。 每个词都映射到精确的时间戳——播放时实时高亮，点击即跳转， 删除区间内的文字自动标记为删除线样式。
文字与画面，终于说同一种语言。
// ASR 数据结构
interface AsrWord {
  text: string
  start: number // 精确到毫秒
  end: number
}

// 词级别交互
onClick(word) → seek(word.start)
onPlay(t) → highlight(word)
onDelete(range) → strikethrough(words)
